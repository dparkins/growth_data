Here is some code to use redhsift-space distortion growth of structure data with the cosmological analysis package cosmomc (http://cosmologist.info/cosmomc/). You will need to download a version of cosmomc first, and then use the updated files in this repository to recompile the source code.

Data is taken from Blake et al 2012 (WiggleZ), Beutler et al 2012 (6dFGRS) and Reid et al (2012) (BOSS).
